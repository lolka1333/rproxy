# ═══════════════════════════════════════════════════════════════════════════
#  blocked.txt — example soft-block list for rproxy
#
#  Activate by setting `blocking = on` in rproxy.conf (the list is already
#  wired there via `blocklist = blocked.txt`).
#
#    • one domain per line; matching is subdomain-aware and case-insensitive
#      (doubleclick.net also blocks stats.g.doubleclick.net)
#    • "#" starts a comment; blank lines are ignored
#    • a leading "*." or "." is optional (ads.example == *.ads.example)
#
#  Blocked sites are NOT hard-failed: rproxy relays up to `block-cap` bytes
#  (13 KiB by default), then drops the connection.
#
#  This is a STARTING POINT of well-known ad / tracker / telemetry domains —
#  curate it for your needs. Note: traffic over QUIC/HTTP-3 (UDP 443) or from
#  apps that ignore the system proxy won't pass through rproxy and so can't be
#  blocked here (see the README).
# ═══════════════════════════════════════════════════════════════════════════
